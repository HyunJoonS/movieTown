기능
---

- **TMDB API**를 이용한 영화 정보 제공
    - 영화, 영화인 검색 및 평가 기능
- **Socket.io** 를 이용한 **실시간 채팅** 기능
- 게시판 구현
    - 기본적인 **CRUD**, **페이징**,  **댓글**, **대댓글** 기능
- **google, facebook 소셜 로그인 기능(Oauth2.0)**
- **passport 사용자 인증 토큰** 관리
- 프로필 관리 기능 (사진, 닉네임, 비밀번호)

- ubuntu 20.04
     - 개인 서버 구축 및 운영
